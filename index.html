About shoppingbasket

Application to price a basket of goods.

Author

Mike Thomas (mikepthomas@outlook.com)

Building

Build

To make shoppingbasket.jar, install Maven and run the following command:

mvn clean install

Test

To run all unit-tests, run the following command:

mvn test

Documentation

To generate reports and documentation, run:

mvn site

Running the application

Deploy the war file to application server and then visit:

http://localhost:8080/shoppingbasket
